The Parent Name fields can be populated by any name of either parent. For example you can enter the surname of one parent in the first field and only a forename of the other parent in the second field and only matches containing those parent names will be returned. Please note that these indexes generally contains a father's name and a mother's name, but not always.

Please also note that some Catholic parish registers record forenames in Latin.

The indexes available on this site include Latin to English translation and you need only enter the forename in English. Your search results will return the English name followed by the Latin equivalent in brackets ‘()’ where this occurs.
