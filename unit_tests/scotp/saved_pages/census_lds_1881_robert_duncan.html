Search Options

Enter the forename of another person you might expect to find in the household or street to narrow your search.

Please note that this search looks for names on the same census page. As a result, it finds matches for people who are on that census page but possibly not in the same household. It will not find a match if the household is over two pages and the names are on separate pages.

For help on choosing the right forename search option read our guidance on forenames

Search Options
Gender:

The default is both.

Select male or female to narrow your search.

Enter an age range to narrow your search.

In 1841 ages over 15 years were rounded down to the nearest five, for example, 29 was recorded as 25. Exact ages were given for those under 15 years old.

If no age was given the enumerator entered NK (not known) in the age column. If an indexer couldn’t read the age it is shown as 131 or "U".

If you enter an age in the first box only all ages from the specified age will be included. If you enter an age in the second box only all ages to the specified age will be included. If you leave both blank all ages will be included.

Select a county or city to narrow your search.

You can also select shipping as an option from the 1861 census onwards.

 

 

 

Displays all registration districts in Scotland unless a county, city or shipping has been selected.

If you select a county or city, the districts for that county or city are displayed. Select a district to narrow your search further.

Please note that not all districts are valid for all years. If a search returns no results for a specific district, try widening the search to 'All Districts' for the county.

Enumerators recorded some addresses in abbreviated form, for example, St. for Street.

Enter the name of the house or street to narrow your search. For example, a search on "Register" will return entries for “East Register Street”, “West Register St”, “Register Place” and so on but a search on "Register Street" will not return results for "Register St".

If you know where the person lived around the time of the 1881 census enter the county or district name to narrow your search.

County names do not include "shire", for example, to search for entries in the county of Perthshire enter Perth.

The following old county names were still in use in 1881: Edinburgh (Midlothian), Elgin (Moray), Forfar (Angus), Haddington (East Lothian) and Linlithgow (West Lothian).
