The only purpose of this app is to install the WikiTree Sourcer browser extension in Safari. To use the extension go to Safari.

You can turn on the WikiTree Sourcer Safari extension in Settings > Safari > Extensions or through Manage Extensions in Safari.

You can turn on WikiTree Sourcer’s extension in Safari Extensions preferences.

WikiTree Sourcer’s extension is currently on. You can turn it off in Safari Extensions preferences.

WikiTree Sourcer’s extension is currently off. You can turn it on in Safari Extensions preferences.
